## Mathematical Notation

`¬` = __Not__

`϶` = __Such that__

`∈` = __Element of__  
Could be used like, `a ∈ b`, meaning `a` is an element of `b`. This kind of usage belongs to set theory.

`∴` = __Therefore__

`∵` = __Because__
